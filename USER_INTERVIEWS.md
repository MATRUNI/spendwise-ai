# User Interviews & Validation

## Interview 1: Lokesh Mawar
**Role:** Full Stack Developer (FSD)  
**Company:** Consult Quark  

### The Feedback
> "I found this app to be good for anyone struggling to track their AI costs. It is a helpful tool for managing both subscriptions and usage-based expenses in one place. Adding a feature to track the long-term history of expenses would make it even better for spotting trends over time. Overall, it is a good startup that addresses our expenditures."

### Key Takeaways
1.  **Validation of Usage-Based Tracking**: The engine's ability to handle both seats (subscriptions) and API usage (usage-based) was specifically called out as helpful.
2.  **The "History" Gap**: There is a clear desire for historical data. While our current audit is a "point-in-time" snapshot, users want to see how their spend evolves over months.
3.  **Professional Utility**: The tool was recognized as a legitimate "startup" solution for managing consultancy-level expenditures.
